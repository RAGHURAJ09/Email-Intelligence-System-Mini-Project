An **AI + Machine Learning–based web application** that intelligently analyzes customer emails to identify **intent**, **sentiment**, and **priority**, and generates **professional automated replies**.

This project demonstrates a **real-world customer support automation system**, suitable for **final-year projects**, **GitHub portfolios**, and **AI/ML demonstrations**.

---

## 🚀 Key Features

- 🔍 **Email Intent Classification** using Machine Learning  
- 😊 **Sentiment Analysis** (Positive / Neutral / Negative)  
- ⚡ **Priority Detection** (High / Medium / Low)  
- 🤖 **AI-Generated Suggested Replies**  
- 📊 **Dashboard with Email History**  
- 🧠 **Hybrid AI System (ML + LLM)**  
- 🌐 **Flask-Based Web Application**  
- 🔐 **Secure API Key Management**

---

## 🧠 How It Works

1. 📩 User submits a customer email  
2. 🧮 ML model predicts the **email intent**  
3. 🤖 AI model analyzes **sentiment & priority**  
4. ✉️ AI generates a **professional reply**  
5. 📊 Results are displayed on the dashboard  

This hybrid approach ensures **fast predictions** and **high-quality responses**.
